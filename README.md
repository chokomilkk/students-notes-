# Students notes

A small learning project for managing notes about students.  
The repository contains a Jupyter Notebook `Students_notes.ipynb` and an Excel file with the list of students.

## Contents

- `Students_notes.ipynb` – main notebook with analysis and notes.
- Excel file with the list of students – source data for the notebook.

## How to run

1. Install Python and Jupyter Notebook / JupyterLab.
2. Clone this repository or download it as a ZIP archive.
3. Open `Students_notes.ipynb` in Jupyter and run the cells in order.

## Project purpose

This project was created for practicing Python, Jupyter Notebooks, and working with tabular data (Excel) for educational purposes.
